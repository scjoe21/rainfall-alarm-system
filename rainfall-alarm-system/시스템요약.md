# 주민대피 알람 시스템 — 요약

## 1. 개요

**목적**: 호우 시 15분 실측치·1시간 예측치를 기반으로 위험 지역을 감지하고, 주민대피 알람을 실시간으로 제공

**배포**: https://rainfall-alarm-kr.fly.dev/

---

## 2. 알람 발동 조건

| 조건 | 기준값 | 설명 |
|------|--------|------|
| 15분 실측치 | ≥ 20mm | 과거 15분간 누적 강수량 |
| 1시간 예측치 | ≥ 55mm | 향후 1시간 예상 강수량 |

**두 조건을 모두 만족할 때** 알람 발동 → 해당 읍면동 지도 영역 깜빡임 + 알람 목록 표시

---

## 3. 데이터 소스

| 구분 | API | 갱신 | 용도 |
|------|-----|------|------|
| 15분 실측치 | APIHUB nph-aws2_min (RN-15m) | 10분 | AWS 관측소별 15분 강수량 |
| 1시간 실측치 | APIHUB nph-aws2_min (RN-60m) | 10분 | AWS 관측소별 1시간 누적 강수량 |
| 1시간 예측치 | APIHUB nph-dfs_vsrt_grd (VSRT) | 10분 | 레이더 기반 향후 1시간 예측 |
| 폴백 | 공공데이터 초단기실황 (RN1) | - | APIHUB 실패 시 격자별 1시간 강수량 |

**좌표 보완**: nph-aws2_min에 좌표가 없어 `aws-stations-fallback.json` 좌표를 보조로 사용

---

## 4. 스케줄 구조 (3단계)

| 타이머 | 주기 | 역할 |
|--------|------|------|
| Alert Check | 5분(특보 시) / 30분(미발효 시) | 호우특보 발효 여부 확인 |
| Fast Poll | 5분 | 특보 발효 광역 — 35개 AWS 관측소 조건 확인 |
| Slow Poll | 30분 | 특보 미발효 — 전국 관측소 조건 확인 |
| AWS Refresh | 10분 | APIHUB 1회 호출로 전국 실측치·예측치 갱신 |

---

## 5. 아키텍처

```
[클라이언트]                    [서버]
  App.jsx
    ├─ MetroSelector (광역시도 선택)
    ├─ DistrictSelector (기초자치단체 선택)
    ├─ EmdMap (읍면동 지도 + 툴팁)
    └─ AlarmList (알람 내역)

  WebSocket ◄─────────────────── emitAlarm, rainfall_update

[서버]
  scheduler.js
    ├─ runAwsRefresh (10분) → fetchAllAwsData → aws_rainfall 저장
    ├─ runFastPoll (5분) / runSlowPoll (30분) → processAwsStations
    └─ syncAwsToRainfallRealtime → rainfall_realtime, rainfall_forecast

  alarmService.js
    └─ checkAlarmConditionForAwsStation (조건 판정 + DB 저장)

  kmaAPI.js
    ├─ fetchAllAwsData (APIHUB nph-aws2_min)
    ├─ getVsrtForecastHourly (APIHUB nph-dfs_vsrt_grd)
    └─ getAWSRealtime15min (공공 초단기실황, 폴백용)

[Cloudflare Worker] — 한국 PoP 경유
  /kma-public/*  → 공공데이터포털
  /kma-apihub/*  → 기상청 API허브
```

---

## 6. 주요 테이블

| 테이블 | 용도 |
|--------|------|
| aws_rainfall | AWS 관측소별 15분/1시간 실측치, 1시간 예측치 |
| rainfall_realtime | weather_stations 기준 15분/1시간 실측치 (읍면동 맵용) |
| rainfall_forecast | weather_stations 기준 1시간 예측치 |
| aws_alarm_logs | 알람 발동 이력 |
| emds, districts, metros | 행정구역 (읍면동·기초자치단체·광역시도) |

---

## 7. 환경 변수

| 변수 | 필수 | 설명 |
|------|------|------|
| KMA_API_KEY | ✅ | 공공데이터포털 인증키 |
| KMA_APIHUB_KEY | ✅ | 기상청 API허브 인증키 |
| CLOUDFLARE_WORKER_URL | ✅ | Worker 프록시 URL (한국 IP 경유) |
| CLOUDFLARE_PROXY_TOKEN | - | Worker 인증 토큰 |
| KMA_DAILY_LIMIT | - | 일일 API 한도 (기본 100만) |

---

## 8. 지도 표시 항목

읍면동 영역 터치 시 툴팁:

- **1시간 실측치**: RN-60m (mm)
- **15분 실측치**: RN-15m (mm)
- **1시간 예측치**: VSRT 예측 (mm/hr)

---

## 9. 진단 API

- `/api/status` — 설정·DB 상태
- `/api/status?test=1` — 공공 API 연결 테스트
- `/api/api-usage` — 일일 API 호출량
- `/api/debug/aws-parse` — APIHUB 원시 응답·파싱 결과
